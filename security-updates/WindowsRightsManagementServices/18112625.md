密码箱
======

密码箱驻留在客户端计算机上。密码箱是标识 RMS 可信的计算机或设备的一个不可或缺的组成部分。计算机或设备在安装并激活 RMS 客户端时接收到密码箱。每个密码箱都建立在硬件标识符的基础上，因此密码箱不仅是唯一的，而且绑定到特定的计算机上。密码箱包含已激活计算机的私钥。

计算机无需成为 Active Directory 域的成员即可接收密码箱或计算机证书。
