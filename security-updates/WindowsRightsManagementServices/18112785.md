根认证服务器注册
================

部署的 RMS 系统中的第一台服务器需要通过 Microsoft 注册服务进行注册。该服务器为根认证服务器，当其与 Internet 连接时可在设置过程中自动注册，或当该服务器断开与网络的连接时可手动注册。有关手动注册服务器的详细信息，请参阅本文档集中的“运行 RMS 服务器”中的“手动注册根认证服务器”。

注册请求需要下列输入参数：

-   1024 位公钥。该公钥为 RMS 公钥。
-   要进行注册的 RMS 服务器的版本、名称和 URL。

Microsoft 注册服务将上述信息仅用于创建服务器许可方证书之目的，而且存储这些信息的目的也只有一个，即供吊销之用。

Microsoft 注册服务返回一个包含注册服务器的许可方证书链的证书链，以及由该注册服务器签署的证书。该证书包含使用注册私钥签署的服务器公钥以及已注册服务器的版本和 URL。该证书不仅可以授予根认证服务器向授权服务器颁发服务器许可方证书的权限，还可以授予其颁发权限帐户证书、客户端许可方证书、发布许可证以及用户许可证的权限。

服务器许可方证书的有效期为一年。有效期自颁发证书之日算起。有效期届满时，可以对该证书进行续订。该服务器颁发的证书和许可证的有效期为七年。有效期自颁发证书或许可证之日算起。

有关吊销证书的信息已按照注册请求中的说明添加到服务器许可方证书中。Microsoft 注册服务的公钥作为吊销密钥添加到该证书中。此外，如果指定了第三方吊销密钥，则还可以将其作为吊销密钥添加到该证书中。
