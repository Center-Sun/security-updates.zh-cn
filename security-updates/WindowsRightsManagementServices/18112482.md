从服务器上删除 Microsoft SQL Server 2000 Desktop Engine (MSDE 2000)
===================================================================

从服务器中删除 MSDE 2000
------------------------

使用下列步骤从服务器中删除 MSDE 2000：

#### 从服务器中删除 MSDE 2000

1.  以本地管理员身份登录到计算机

2.  单击“开始”，指向“控制面板”，然后单击“添加或删除程序”。

3.  单击“更改或删除程序”。

4.  单击 Microsoft SQL Server 2000 Desktop Engine 程序，然后单击“删除”。
