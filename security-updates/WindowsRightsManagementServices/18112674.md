打开或关闭日志记录
==================

要执行该过程，必须使用正在访问的计算机上是“管理员”组成员的域用户帐户从本地登录到管理网站。“Domain Admins”组成员也可以执行该过程。作为一种最佳安全设置，可以考虑使用“**运行方式**”选项来执行该过程。

要打开“**全局管理**”页面，单击“**开始**”，依次选择“**所有程序**”、“**Windows RMS**”，然后单击“**Windows RMS 管理**”。

在打开日志记录之前，请确保 RMS 服务器与数据库服务器相连接并且已启动数据库服务。如果 Message Queuing 无法将日志传送到日志数据库，它会将数据以队列形式保存在 RMS 服务器的硬盘中。这一操作将持续到服务器的存储空间满为止。这种情况下 RMS 不会显示错误，因为此功能旨在与 SQL 服务器的连接中断时支持日志记录。

打开或关闭日志记录
------------------

#### 打开或关闭日志记录

1.  打开“**全局管理**”页面，然后在要打开或关闭日志记录的网站旁边，单击“**在此网站上管理 RMS**”。

2.  在“**管理链接**”区域中，单击“**日志设置**”。

3.  在“**日志服务器和数据库**”区域中，选中“**打开日志记录**”复选框，然后单击“**更新**”。

    要关闭日志记录，清除该复选框，然后单击“**更新**”。

有关执行该过程的详细信息，请参阅本主题中前面的“[打开和关闭日志记录](https://technet.microsoft.com/50ccd827-2d39-41e7-a395-3d5f5836869b)”。
