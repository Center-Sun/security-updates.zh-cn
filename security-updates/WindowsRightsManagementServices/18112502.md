RMS 计算机证书
==============

RMS 计算机证书可以标识 RMS 系统信任的计算机或设备。

在激活客户端的过程中，计算机或设备会生成 RMS 计算机证书。

RMS 计算机证书包含已激活计算机的公钥。此证书由包含在 Service Pack 1 的 RMS 客户端中的 Microsoft 激活密钥签署。

计算机无需成为 Active Directory 域的成员即可接收密码箱或计算机证书。
